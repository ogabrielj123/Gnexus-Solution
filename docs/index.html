Stack Tecnológico

Construído com as melhores tecnologias modernas

Backend

Node.js 24 com Express 5, PostgreSQL + Drizzle ORM, TypeScript, autenticação por sessão e certificados digitais via node-forge

Frontend

React 19 com Vite, TailwindCSS, shadcn/ui, React Query para gerenciamento de estado e Zod para validação

Infraestrutura

pnpm Workspaces para monorepo, TypeScript 5.9, OpenAPI/Orval para codegen, esbuild para build otimizado

eSocial

Integração com webservice eSocial via SOAP, geração de XML com assinatura digital, suporte a ambientes de homologação e produção

Banco de Dados

PostgreSQL com schema multi-cliente, versionamento via Drizzle migrations, backup automático em produção

CI/CD

GitHub Actions para testes, build e deploy automático, monitoramento de performance e alertas de erro
